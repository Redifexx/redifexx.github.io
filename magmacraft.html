MagmaCraft

A multiplayer voxel sandbox featuring procedural terrain generation.

  • Category Software Development
  • Core Tech OpenGL 4.6, ENet
  • Project Date December 2025 - Current
  • Project Site GitHub

Overview

Magma Craft represents a complete architectural overhaul of my previous voxel-based project, designed to rectify fundamental design flaws and integrate multiplayer networking. Built upon my custom Magma framework, this project serves as a stress test for my engine development skills. I successfully engineered a custom networking layer for packet management, a serialization system for world persistence, a streaming terrain system, and multithreaded chunk generation. I am currently expanding the project's capabilities, with a roadmap that includes real-time world modification, procedural decoration, and a deferred rendering pipeline.
